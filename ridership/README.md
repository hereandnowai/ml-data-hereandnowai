# Chicago Transit Ridership Dataset

### *AI is Good*

Detailed daily boarding totals for the Chicago Transit Authority (CTA), ideal for time-series forecasting and urban mobility analysis.

## 🚌 Dataset Source

This dataset was created by **Chicago's Transit Authority (CTA)**. You can access the official data via the [Chicago Data Portal](https://homl.info/ridership).

## 📈 Analysis Goals

- Forecast seasonal transit demand.
- Analyze the impact of external events on public transportation.
- Optimize route planning based on historical boarding trends.

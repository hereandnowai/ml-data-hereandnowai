# ML Data Repository

### *AI is Good*

A collection of open datasets curated for machine learning development. This repository serves as a centralized hub for various datasets used in educational and research projects, ensuring easy access and clear documentation.

Each directory contains a specific dataset, along with information about its source, description, and licensing.

## 📂 Project Structure

- [housing/](housing/): California housing prices dataset.
- [images/](images/): Sample images for computer vision tasks.
- [inception/](inception/): ImageNet class names and metadata.
- [jsb_chorales/](jsb_chorales/): JSB Chorales dataset in CSV format.
- [lifesat/](lifesat/): OECD Better Life Index and GDP per capita data.
- [ridership/](ridership/): Chicago Transit Authority (CTA) daily boarding totals.
- [titanic/](titanic/): The classic Titanic survivor dataset.
- [videos/](videos/): Sample videos for video processing tasks.

# Titanic: Machine Learning from Disaster

### *AI is Good*

The classic Titanic dataset for binary classification, used to predict which passengers survived the shipwreck based on features like age, gender, and socio-economic class.

## 🛳 Dataset Overview

- **Task**: Binary Classification (Survived/Did not survive).
- **Features**: `Pclass`, `Sex`, `Age`, `SibSp`, `Parch`, `Fare`, `Embarked`.
- **Target**: `Survived`.

## 📁 Included Files

- `train.csv`: Training set with labels.
- `test.csv`: Test set for model evaluation (no labels).

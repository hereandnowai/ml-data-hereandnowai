# ImageNet Class Labels & Metadata

### *AI is Good*

This directory contains metadata and class names from the ImageNet database, primarily used for mapping model output indices to human-readable category names.

## 🏷 Dataset Information

- **Source**: [ImageNet Official Site](https://www.image-net.org/)
- **Core Reference**: Russakovsky et al., "ImageNet Large Scale Visual Recognition Challenge," *IJCV*, 2015.
- **Resources**: [Paper](http://link.springer.com/article/10.1007/s11263-015-0816-y), [ArXiv](http://arxiv.org/abs/1409.0575).

## 📂 Included Data

- **Labels**: Mappings for the ILSVRC dataset.
- **Metadata**: Attribute annotations and bibtex references.
